Sky Aesthetic

SkyAesthetics is a React Native mobile application that streamlines clinic workflows end to end: appointment booking, chart-based patient intake, and appointment history. Users pick a service, fetch available consultant time slots from the API, and create appointments using a Formik-powered booking flow. After booking, the app renders dynamic medical/clinical charts driven by server templates with a template-cached preview mode. Charts support many field types (text inputs, checkboxes, dropdowns, vitals, optical measurements, ranges, smart options, spine selection, multiple image inputs like body/side-by-side/files/sketch, and signature capture) and can be filled as “Fill now” or “Fill at premises”. Filled results are included in the `chart_submissions` payload during appointment creation, so clinicians and users see the same chart data instantly. API calls use Redux Toolkit Query, and the app uses Firebase Cloud Messaging for push notifications, including payment completion handling to close the in-app payment web view.

  1. React Native
  2. TypeScript
  3. Redux Toolkit Query (RTK Query)
  4. Formik
  5. Firebase Cloud Messaging
  6. react-native-image-picker
  7. react-native-inappbrowser-reborn
